## 📖 Quick Start Guide

### 1. Backup Your Current Settings
```
Use any tool: CHIRP, UVTools2, Multi-UVTools, or Quansheng Software
- Connect radio → Backup → Save .img or .DAT file
- Store safely for recovery if needed
```

### 2. Flash ApeX v7.6.5br5
```
- Download firmware from repository
- Use your preferred programming tool
- Flash to radio (DFU mode)
- Boot and enjoy automatic migration!
```

### 3. Verify Migration
```
- Check SysInf for imported battery diagnostics
- Browse channels - your settings should be there
- If needed, restore from backup
```
