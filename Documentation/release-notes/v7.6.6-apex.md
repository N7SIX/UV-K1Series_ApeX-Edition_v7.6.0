# Release Notes - v7.6.6 ApeX Edition

Release date: March-April 2026
Build target: UV-K1 Series / UV-K5 V3
Edition: ApeX (all-in-one)

## Release Summary
v7.6.6 introduced broad architecture and runtime reliability improvements, including event-system groundwork and multiple spectrum and safety hardening updates.

## Compatibility
- Supported: UV-K1 and UV-K5 V3 hardware with PY32F071 MCU
- Not supported: UV-K5 V1 or non-PY32F071 variants

## Key Updates
- Event-driven architecture framework and handler integration groundwork.
- Spectrum reliability refinements for sweep integrity, RSSI handling, and control consistency.
- NOAA enablement and key-map conflict cleanup in ApeX profile.
- Low-voltage TX/persistence protections and deferred-save behavior.
- Native full-block spectrum persistence and transactional A/B settings snapshot recovery.

## Validation Status
- Build path: repository Docker build workflow (ApeX preset).
- Runtime behavior validated through source diagnostics and release integration checks for documented safety and persistence changes.

## Upgrade Notes
- Back up calibration before flashing.
- Confirm battery calibration and key workflows (TX, scan, NOAA toggle, spectrum controls) after upgrade.

## Outcome
- More predictable behavior in weak-power conditions.
- Improved maintainability and clearer internal module responsibilities.
