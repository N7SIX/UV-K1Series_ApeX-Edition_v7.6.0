# Release Notes - v7.6.5 ApeX Edition

Release date: March 2026
Build target: UV-K1 Series / UV-K5 V3
Edition: ApeX (all-in-one)

## Release Summary
v7.6.5 delivered major spectrum analyzer enhancements, rendering improvements, and user-experience refinements in ApeX all-in-one firmware.

## Compatibility
- Supported: UV-K1 and UV-K5 V3 hardware with PY32F071 MCU
- Not supported: UV-K5 V1 or non-PY32F071 variants

## Key Updates
- Professional-grade spectrum visualization with waterfall and smoothing refinements.
- Improved trigger/squelch behavior and adaptive display dynamics.
- Expanded persistence and state-restoration behavior for spectrum settings.
- Performance-focused rendering and update-path optimizations for responsiveness.
- Airband (AM) restoration after Spectrum/Waterfall: fixed a bug where AM reception could be lost after using spectrum features. All VFO state is now fully restored on exit.
## Validation Status
- Build and integration were validated for this release line.

## Upgrade Notes
- Back up calibration before flashing.
- Validate primary spectrum controls and saved settings behavior after first reboot.

## Outcome
- Better real-time usability, stronger visual clarity, and improved operational confidence during scan and listen workflows.
