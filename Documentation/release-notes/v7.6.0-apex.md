# Release Notes - v7.6.0 ApeX Edition

Release date: Early 2026
Build target: UV-K1 Series / UV-K5 V3
Edition: ApeX (all-in-one)

## Release Summary
v7.6.0 established the ApeX all-in-one foundation and integrated broad feature coverage into a single production firmware profile.

## Compatibility
- Supported: UV-K1 and UV-K5 V3 hardware with PY32F071 MCU
- Not supported: UV-K5 V1 or non-PY32F071 variants

## Key Updates
- Consolidated multi-profile capabilities into ApeX edition.
- Expanded spectrum, utility, and workflow feature availability.
- Established baseline architecture for later safety and reliability hardening.

## Validation Status
- Initial ApeX baseline build and integration checks were completed for release readiness.

## Upgrade Notes
- Back up calibration before flashing.
- Confirm baseline operation (VFO, channel mode, scan, and display) after first boot.

## Outcome
- Single-feature-rich build path with unified user experience and easier maintenance direction.
